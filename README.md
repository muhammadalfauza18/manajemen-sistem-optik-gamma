# 👓 Manajemen Sistem Optik Gamma

![PHP](https://img.shields.io/badge/PHP-8.2-blue?logo=php)
![MySQL](https://img.shields.io/badge/MySQL-Database-orange?logo=mysql)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5-purple?logo=bootstrap)
![License](https://img.shields.io/badge/Status-Active-success)

Sistem informasi berbasis web untuk membantu proses administrasi pada **Optik Gamma**. Aplikasi ini dikembangkan untuk mempermudah pengelolaan data pasien, rekam medis, transaksi penjualan, produk, serta laporan secara terintegrasi.

---

## 🌐 Demo Website

**Website :**
> https://optikgamma.infinityfreeapp.com

---

# 📌 Fitur Utama

- 🔐 Login Multi User (Admin & Karyawan)
- 👤 Manajemen Data Pasien
- 🩺 Manajemen Rekam Medis
- 👓 Manajemen Produk
- 💳 Manajemen Transaksi
- 📊 Dashboard Statistik
- 📄 Laporan Penjualan
- 📥 Export Laporan ke PDF
- 📈 Grafik Penjualan
- 👨‍💼 Manajemen Data Karyawan
- 🖼 Upload Foto Profil

# 📊 Dashboard

Dashboard menampilkan informasi secara real-time seperti:

- Total Pasien
- Total Produk
- Total Transaksi
- Pendapatan
- Grafik Penjualan
- Pasien Terbaru
